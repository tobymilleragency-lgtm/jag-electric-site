## 4. SCHEMA NOTES

### LocalBusiness (homepage and all city pages)

```json
{
  "@context": "https://schema.org",
  "@type": ["LocalBusiness", "Electrician"],
  "name": "JAG Electric, LLC",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "807 E 7th St",
    "addressLocality": "Galena",
    "addressRegion": "KS",
    "postalCode": "66739",
    "addressCountry": "US"
  },
  "telephone": "(620) 714-1125",
  "email": "jag.electric@yahoo.com",
  "url": "https://[DOMAIN]/",
  "areaServed": [
    {"@type": "City", "name": "Pittsburg", "addressRegion": "KS"},
    {"@type": "City", "name": "Galena", "addressRegion": "KS"},
    {"@type": "City", "name": "Frontenac", "addressRegion": "KS"},
    {"@type": "City", "name": "Girard", "addressRegion": "KS"},
    {"@type": "City", "name": "Joplin", "addressRegion": "MO"}
  ],
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.9",
    "reviewCount": "50",
    "bestRating": "5"
  },
  "openingHoursSpecification": [
    {
      "@type": "OpeningHoursSpecification",
      "dayOfWeek": ["Monday","Tuesday","Wednesday","Thursday","Friday"],
      "opens": "08:00",
      "closes": "17:00"
    }
  ],
  "priceRange": "$$",
  "sameAs": [
    "https://www.facebook.com/jagelectricks/"
  ]
}
```

**Notes:**
- Use @type array with both "LocalBusiness" and "Electrician" — Electrician is a valid Schema.org type and helps category signals.
- reviewCount: use 50 from confirmed GBP count. If site launches and count has changed, update.
- openingHours: use conservative 8-5 M-F. Do NOT add 24-hour or "emergency" hours unless Joe confirms. Facebook mentions limited evening/weekend — that is a phone-call topic, not a schema field.
- Remove or leave blank license fields — do NOT invent a license number.

### Service pages — use Service schema

Each /services/* page should have a Service schema block:

```json
{
  "@context": "https://schema.org",
  "@type": "Service",

  "provider": {"@id": "https://[DOMAIN]/#business"},
  "areaServed": {"@type": "State", "name": "Kansas"},
```
